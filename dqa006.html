Available Q4OS Applications

Comprehensive guide to software available for Q4OS Trinity desktop.

Pre-installed Applications

Q4OS comes with essential applications pre-installed for immediate productivity:

System & Desktop

  • Konqueror: File manager and web browser
  • Konsole: Terminal emulator
  • Trinity Control Center: System configuration
  • Kate: Advanced text editor
  • KWrite: Simple text editor
  • Ark: Archive manager

Internet & Communication

  • Firefox: Web browser
  • Thunderbird: Email client (available for installation)
  • KMail: Trinity email client
  • Pidgin: Multi-protocol instant messenger

Multimedia

  • VLC Media Player: Universal media player
  • Amarok: Music player and library manager
  • KMix: Audio mixer
  • KsCD: CD player

Office & Productivity

Office Suites

  • LibreOffice: Complete office suite
    • Writer - Word processor
    • Calc - Spreadsheet application
    • Impress - Presentation software
    • Draw - Vector graphics
    • Base - Database management
    • Math - Formula editor
  • OnlyOffice: Microsoft Office compatible suite
  • WPS Office: Alternative office suite
  • AbiWord: Lightweight word processor
  • Gnumeric: Lightweight spreadsheet

PDF & Document Viewers

  • Okular: Universal document viewer
  • Evince: PDF and document viewer
  • KPDF: Trinity PDF viewer

Note Taking & Organization

  • KNote: Desktop sticky notes
  • Basket: Note organization tool
  • CherryTree: Hierarchical note taking
  • Zim: Desktop wiki

Graphics & Multimedia

Image Editing

  • GIMP: Professional image manipulation
  • Krita: Digital painting and illustration
  • Inkscape: Vector graphics editor
  • KolourPaint: Simple paint program
  • KSnapshot: Screen capture tool
  • Gwenview: Image viewer

Video & Audio

  • Kdenlive: Non-linear video editor
  • OpenShot: Easy video editor
  • Audacity: Audio editing and recording
  • K3b: CD/DVD burning software
  • Brasero: Disc burning utility
  • DeaDBeeF: Lightweight music player

3D & CAD

  • Blender: 3D creation suite
  • FreeCAD: Parametric 3D modeler
  • LibreCAD: 2D CAD application
  • SketchUp: 3D modeling (via Wine)

Development Tools

Code Editors & IDEs

  • Visual Studio Code: Modern code editor
  • KDevelop: Integrated development environment
  • Qt Creator: Qt application development
  • Eclipse: Java IDE and platform
  • Code::Blocks: C/C++ IDE
  • Geany: Lightweight IDE

Programming Languages

  • GCC: GNU Compiler Collection
  • Python: Python interpreter and libraries
  • Node.js: JavaScript runtime
  • Java: OpenJDK Java development kit
  • Git: Version control system
  • SVN: Subversion version control

Database Tools

  • MySQL Workbench: MySQL administration
  • pgAdmin: PostgreSQL administration
  • SQLite Browser: SQLite database browser
  • DBeaver: Universal database tool

Internet & Communication

Web Browsers

  • Firefox: Mozilla web browser
  • Chromium: Open-source Chrome
  • Opera: Feature-rich browser
  • Vivaldi: Customizable browser
  • Konqueror: Trinity web browser

Email & Messaging

  • Thunderbird: Full-featured email client
  • Evolution: Groupware suite
  • Claws Mail: Lightweight email client
  • Telegram: Secure messaging
  • Discord: Voice and text chat
  • Signal: Private messenger

File Sharing & Cloud

  • Dropbox: Cloud storage service
  • Nextcloud: Self-hosted cloud
  • Syncthing: Peer-to-peer sync
  • FileZilla: FTP client
  • Transmission: BitTorrent client

Gaming & Entertainment

Gaming Platforms

  • Steam: Digital game distribution
  • Lutris: Gaming platform manager
  • PlayOnLinux: Wine frontend for games
  • GameHub: Unified gaming library

Classic Games

  • KMahjongg: Mahjongg solitaire
  • KMines: Minesweeper clone
  • KSolitaire: Card games collection
  • KAsteroids: Asteroids game
  • Chess: Various chess programs

Emulators

  • RetroArch: Multi-system emulator
  • MAME: Arcade machine emulator
  • DOSBox: DOS games emulator
  • PCSX2: PlayStation 2 emulator

System Utilities

System Monitoring

  • System Monitor: Task manager
  • htop: Interactive process viewer
  • Baobab: Disk usage analyzer
  • GParted: Partition editor
  • Hardware Lister: System information

Backup & Recovery

  • Timeshift: System snapshots
  • Rsync: File synchronization
  • Deja Dup: Simple backup tool
  • Back In Time: Snapshot backups

System Cleaning

  • BleachBit: System cleaner
  • Stacer: System optimizer
  • Ubuntu Cleaner: System maintenance

Installation Methods

Graphical Installation

  • Software Centre: User-friendly app installer
  • Synaptic: Advanced package management
  • Q4OS Software Installer: One-click installers for popular apps
  • AppImage: Portable application format
  • Flatpak: Universal package system
  • Snap: Canonical's package system

Command Line Installation

Use APT package manager for command-line installation:

sudo apt update
sudo apt install application-name

Third-party Software

  • Download .deb packages from official websites
  • Add external repositories for specialized software
  • Use Wine for Windows applications
  • Compile from source code when needed

Recommended Essential Applications

Must-Have Applications

  • GIMP (image editing)
  • VLC Media Player (multimedia)
  • Firefox (web browsing)
  • Thunderbird (email)
  • Visual Studio Code (text editing)

System Essentials

  • BleachBit (system cleaning)
  • Synaptic (package management)
  • FileZilla (FTP client)
  • Ark (archive manager)
