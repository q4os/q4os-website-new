Commercial Support

Get professional support for your business. We are ready to make any kind of system customization, including core level API programming as well as user interface modifications. Any form of commercial support ask at support@q4os.org.

Partnership Opportunities

We are currently seeking business partners to accelerate development in order to build the perfect operating system for professional use. If you would like to participate, fund or make an investment in Q4OS, please contact us at partner@q4os.org.

Applications development

Q4OS provides comprehensive development tools and documentation for creating custom applications. Please read the developer manuals in the Documentation section first. We offer complete support for independent developers via e-mail develop@q4os.org or our public developer forum.

Bugs

Please report bugs at our bug report tracker . You are welcome to submit a proposals for Q4OS improvements or new features as well.

Participate

If you would like to help us, contribute or anyhow participate in the development of Q4OS ecosystem, contact us at devteam@q4os.org.
